# 🎤 Studdy Buddy

Your AI-powered study companion with real-time voice conversations.

## Features
- 🎤 Voice input (speech-to-text)
- 🧠 Claude AI responses
- 🔊 Voice output (text-to-speech)
- 🎨 Beautiful dark UI
- ⚡ Real-time conversations

## Tech Stack
- Next.js 16
- Claude API (Anthropic)
- Web Speech API
- Tailwind CSS
